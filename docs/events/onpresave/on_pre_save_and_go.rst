OnPreSaveAndGo
==============
`Data Source`_

.. include:: /includes/not_finished.rst

Сохраняет запись текущего объекта во временную таблицу и загружает другой объект по переданному Id.

Вызывается из шаблона
---------------------

Шаблоны редактирования объектов, на которых возможны кнопки Prev/Next.

Вызывается из событий
---------------------

Не вызывается из событий.

Входные параметры
-----------------

+----------------------+--------------------------------------------------------------+
| название             | описание                                                     |
+======================+==============================================================+
| .. config-property:: | ``ID`` той записи, которую необходимо сохранить.             |
|    :name: id         |                                                              |
|    :type: int        |                                                              |
+----------------------+--------------------------------------------------------------+
| .. config-property:: | ``ID`` той записи, к которой нужно перейти после сохранения. |
|    :name: GoId       |                                                              |
|    :type: int        |                                                              |
+----------------------+--------------------------------------------------------------+

Вызывает события
----------------

- :doc:`/events/onpresave/on_pre_save`

Потенциальное применение
------------------------

Применяется при редактировании нескольких записей, при перемещении по которым используются кнопки Prev/Next.

Ограничения
-----------

Нет ограничений.
