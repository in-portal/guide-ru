OnPreSavePopup
==============
`Data Source`_

.. include:: /includes/not_finished.rst

Сохраняет содержимое вплывающего окна (popup) в соответствующую объекту
:doc:`временную таблицу </database/using_temporary_tables>`. После чего
всплывающее окно (popup) закрывается.

Вызывается из шаблона
---------------------

- ``in-commerce/admin_templates/products/shipping_options.tpl``

Вызывается из событий
---------------------

Не вызывается из событий.

Вызывает события
----------------

- :doc:`/events/onpresave/on_pre_save`

Потенциальное применение
------------------------

Используется, когда требуется сохранить редактируемую во вплывающем окне (popup) запись во временную таблицу.

Ограничения
-----------

Нет ограничений.

Заметки редактора
-----------------

- `Правила <IntechnicWiki:Как_написать_статью#Статьи_с_описанием_событий>`__ необходимо прочесть до того, как начинать
  работу над статьёй.
- Рекомендую **просмотреть уже написанные и принятые статьи** с описанием событий за примером того, как нужно писать.

+-----------------------------------------------------------+--------------------------------------------------------------------------------------------------------+
| текст                                                     | коррекция                                                                                              |
+===========================================================+========================================================================================================+
| Сохраняет содержимое ``Pop-Up``                           | Предложение должно быть завершённым самостоятельным предложением.                                      |
+-----------------------------------------------------------+--------------------------------------------------------------------------------------------------------+
| event (object)                                            | * Нужно читать и соблюдать `правила <IntechnicWiki:Как_написать_статью#Статьи_с_описанием_событий>`__. |
+-----------------------------------------------------------+--------------------------------------------------------------------------------------------------------+
| in-commerce/admin_templates/products/shipping_options.tpl | Не достаточно просто привести шаблон, нужно также об этом красиво                                      |
|                                                           | написать. Чтобы статья не была набором отрывков кода, а была связанным                                 |
|                                                           | текстом.                                                                                               |
+-----------------------------------------------------------+--------------------------------------------------------------------------------------------------------+
| Используется, когда требуется                             | Первое предложение в тексте главы не должно быть продолжением названия                                 |
|                                                           | главы, а должно быть завершённым самостоятельным предложением.                                         |
+-----------------------------------------------------------+--------------------------------------------------------------------------------------------------------+
| Вызывается из событий, Ограничения                        | Если чего-то нету, то зачем создавать главу и писать в ней, что этого нету.                            |
|                                                           | В таком случае главу просто можно не делать.                                                           |
+-----------------------------------------------------------+--------------------------------------------------------------------------------------------------------+
