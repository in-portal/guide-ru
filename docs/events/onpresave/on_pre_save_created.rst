OnPreSaveCreated
================
`Data Source`_

.. include:: /includes/not_finished.rst

Событие создаёт новый объект во временной таблице и сохраняет ``Id`` в сессии и переменных
``Application`` в случае успешного создания.

Вызывается из шаблона
---------------------

Не вызывается из шаблонов.

Вызывается из событий
---------------------

- :doc:`/events/onpresave/on_pre_save`

-----------------

| название              | описание     |
+=======================+==============+
| .. config-property::  | Поля записи. |
|    :name: $items_info |              |
|    :type: array       |              |
+-----------------------+--------------+

Вызывает события
----------------

Не вызывает событий.

Потенциальное применение
------------------------

Внутреннее использование в событии :doc:`/events/onpresave/on_pre_save`.

Ограничения
