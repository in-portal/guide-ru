OnPreSaveAndGoToTab
===================
`Data Source`_

.. include:: /includes/not_finished.rst

Событие сохраняет редактируемый объект и переходит на переданную вкладку (tab).

Вызывается из шаблона
---------------------

Все шаблоны, которые задействованы в редактировании объектов и позволяют осуществлять переход между вкладками (tabs).

Вызывается из событий
---------------------

Из других событий не вызывается.

Входные параметры
-----------------

+----------------------+--------------------------------------------------------+
| название             | описание                                               |
+======================+========================================================+
| .. config-property:: | Таб, к которому нужно перейти после сохранения записи. |
|    :name: GoTab      |                                                        |
|    :type: string     |                                                        |
+----------------------+--------------------------------------------------------+

Вызывает события
----------------

- :doc:`/events/onpresave/on_pre_save`

Потенциальное применение
------------------------

Шаблоны с несколькими вкладками (tabs), предназначенные для редактирования объектов.

Ограничения
-----------

Нет ограничений.
