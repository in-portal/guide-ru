OnPreSaveAndChangeLanguage
==========================
`Data Source`_

.. include:: /includes/not_finished.rst

Событие сохраняет изменения во временную таблицу, после чего меняет язык.

Вызывается из шаблона
---------------------

На данный момент не вызывается из шаблонов.

Вызывается из событий
---------------------

На данный момент не вызывается из других событий.

Вызывает события
----------------

- :doc:`/events/onpresave/on_pre_save`

Потенциальное применение
------------------------

Сохранение записи во временную таблицу при переключении языка. На данный момент в системе не используется.

Ограничения
-----------

Нет ограничений.
