OnPreSaveSubItem
================
`Data Source`_

.. include:: /includes/not_finished.rst

Событие сохраняет (создаёт или обновляет в зависимости от ситуации) записи ``Sub-Item``.

Вызывается из шаблона
---------------------

Не вызывается из шаблонов.

Вызывается из событий
---------------------

Событие является :doc:`hook </components/unit_configs/hooks>`. На данный момент ни к чему не привязанным.

Вызывает события
----------------

В зависимости от ситуации одно из двух:

- :doc:`/events/live_editing/on_create`
- :doc:`/events/live_editing/on_update`

Потенциальное применение
------------------------

Если на одной форме происходит редактирование как родительского, так и дочернего объектов, событие позволяет
не "потерять" изменения дочернего объекта.

Ограничения
-----------

Нет ограничений.
