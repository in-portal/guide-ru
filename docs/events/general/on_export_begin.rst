OnExportBegin
=============
`Data Source`_

.. include:: /includes/not_finished.rst

Событие ``OnExportBegin`` применяется только в административной консоли и происходит в начале процессов
экспорта/импорта данных, когда собираются с шаблона параметров и проверяются параметры экспорта/импорта.
Если параметры экспорта/импорта заданы неправильно, то приложение остаётся на шаблоне параметров
импорта/экспорта и отображает сообщения об ошибках. Если параметры экспорта/импорта заданы правильно, то
запускается итерационный процесс экспорта/импорта посредством перенаправления на шаблон прогресса.

Вызывается из шаблона
---------------------

Вызывается из шаблона параметров импорта/экспорта.

Вызывается из событий
---------------------

Не вызывается из событий.

Входные параметры
-----------------

Отсутствуют.

Вызывает события
----------------

- :doc:`/events/general/on_before_export_begin` - всегда, сразу после того как соберёт параметры экспорта/импорта
  в массив чтобы передать его в этот вызов.

Потенциальное применение
------------------------

Поскольку событие проверяет параметры экспорта/импорта, это событие можно использовать (переопределять,
или делать hook) в случаях, когда означенные параметры требуют нестандартной, особой проверки.

-----------
