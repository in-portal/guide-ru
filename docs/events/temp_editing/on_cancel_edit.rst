OnCancelEdit
============
`Data Source`_

Данное событие используется для того, чтобы стереть созданные ранее
:doc:`временные таблицы </database/using_temporary_tables>` не сохраняя изменённые в них данные
в оригинальные таблицы. Событие в свою очередь вызывает метод ``kTempTablesHandler::CancelEdit``,
который и стирает созданные ранее временные таблицы. Также данное событие закрывает окно редактирования
после успешного стирания временных таблиц.

Вызывается из шаблона
---------------------

Данное событие вызывается только из шаблонов редактирования данных, в случае, когда для их редактирования
используются :doc:`временные таблицы </database/using_temporary_tables>`.

Потенциальное применение
------------------------

Данное событие можно переопределить с целью выполнения дополнительных действий при отмене редактирования
записей.

Ограничения
-----------

Данное событие может быть использовано только при работе с
:doc:`временными таблицами </database/using_temporary_tables>`, в противном случае произойдёт фатальная
ошибка. Если работа идёт с оригинальными таблицами, то должно использоваться событие
:doc:`/events/live_editing/on_cancel` или :doc:`/events/live_editing/on_go_back`.
