OnReset
=======
`Data Source`_

.. include:: /includes/not_finished.rst

Используется для сброса изменений внесенных в форму редактирования записи. Если форма открыта для ввода новой записи,
то происходит установка значений по умолчанию, для полей етой формы.

Потенциальное применение
------------------------

Для примера использования этого события можно посмотреть формы редактирования Users/Administrators на администраторской
части сайта.

.. _Data Source: http://guide.in-portal.org/rus/index.php/EventHandler:OnReset
