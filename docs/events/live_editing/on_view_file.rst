OnViewFile
==========
`Data Source`_

.. include:: /includes/not_finished.rst

Событие применяется для просмотра файла на административной части сайта. С помощью этого события просматривается
только что закаченный файл, до того момента пока не будет отработано событие :doc:`/events/temp_editing/on_save`.

Вызывается из шаблона
---------------------

Вызов метода производится из шаблона редактирования записи, при использовании блока :ref:`form_control_inp_edit_swf_upload`.

Потенциальное применение
------------------------

Изменив этот метод, можно изменить метод загрузки и показа только, что закачанных файлов.

Ограничения
-----------

При конфигурировании поля в котором будет хранится имя закачиваемого файла и при установке
форматтера ``kUploadFormatter``, нужно обратить внимание на параметр ``'direct_links'=>false/true``.
Если этот параметр имеет значение ``true``, просмотр файла будет осуществляться напрямую. Если параметр
имеет значение ``false``, просмотр будет осуществляться с помощью события ``OnViewFile``.
