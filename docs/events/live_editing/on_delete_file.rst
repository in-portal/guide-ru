OnDeleteFile
============
`Data Source`_

.. include:: /includes/not_finished.rst

Событие применяется для удаления файла на администраторской части сайта.

Вызывается из шаблона
---------------------

Вызов метода производится из шаблона редактирования записи, при использовании блока
:ref:`form_control_inp_edit_swf_upload`.

По усмотрению разработчика может использоваться в других местах формы, где необходимо выполнить удаление файла.

Вызывает события
----------------

Никаких событий не вызывает. Производит запись имени удаляемого файла в сессию и при вызове метода
:doc:`/events/temp_editing/on_save` файл удаляется.

Потенциальное применение
------------------------

Можно использовать для изменение метода удаления файла.

Ограничения
-----------

Фактически файл удаляется в момент удаления :doc:`временных таблиц </database/using_temporary_tables>`.
При использовании метода :doc:`/events/temp_editing/on_save`.
