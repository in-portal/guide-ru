OnUploadFile
============
`Data Source`_

.. include:: /includes/not_finished.rst

Событие применяется для закачки файла на административной части сайта в формах редактирования записи.

Вызывается из шаблона
---------------------

Вызов метода производится из шаблона редактирования записи, при использовании блока
:ref:`form_control_inp_edit_swf_upload`.

.. _Data Source: http://guide.in-portal.org/rus/index.php/EventHandler:OnUploadFile
