OnGoBack
========
`Data Source`_

.. include:: /includes/not_finished.rst

В результате вызова этого события происходит перенаправление на предыдущий шаблон.
Как правило используется на BackEnd для остановки каких либо длительных процессов.

Вызывается из шаблона
---------------------

OnGoBack вызывается из шаблонов для Import Export.

Вызывается из событий
---------------------

Вызывается из события :doc:`/events/general/on_export_cancel`.

Потенциальное применение
------------------------

Это событие можно использовать для выполнения дополнительных действий, необходимых в случае преждевременной
остановки процесса импорта (или экспорта). Дополнительная информация доступна
:doc:`/events/temp_editing/on_cancel_edit` и :doc:`/events/general/on_export_cancel`.

.. _Data Source: http://guide.in-portal.org/rus/index.php/EventHandler:OnGoBack
