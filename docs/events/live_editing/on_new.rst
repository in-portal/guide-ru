OnNew
=====
`Data Source`_

.. include:: /includes/not_finished.rst

Данное событие нужно для того, чтобы можно было заполнить все поля объекта значениями по умолчанию. Событие OnNew
происходит при открытии формы ввода данных для создания новой записи.

Вызывается из шаблона
---------------------

Выполняется:

- при вызове формы для создания новой записи путем передачи параметра `event=OnNew`;
- непосредственно из формы редактирования записи, посредством тэга :doc:`/tags/preset_form_fields`.

Потенциальное применение
------------------------

В качестве примера можно рассмотреть создание новой записи для раздела users на администраторской части сайта.
