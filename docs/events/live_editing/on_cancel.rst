OnCancel
========
`Data Source`_

.. include:: /includes/not_finished.rst

Данное событие используется для того, чтобы закрыть окно редактирования без сохранения изменений.

Вызывается из шаблона
---------------------

Вызывается из PopUps редактирования записей. Или отмены изменений в формах основного экрана. Для примера
можно посмотреть Администраторскую часть сайта на platform Configuration->System Variables.

Вызывается из событий
---------------------

Может вызываться из события :doc:`/events/temp_editing/on_cancel_edit`.

Потенциальное применение
------------------------

Данное событие можно переопределить с целью выполнения дополнительных действий при отмене редактирования записей.

Ограничения
-----------

Не рекомендуется использовать для отмены редактирования записей использующие временные таблицы. Для этого
используется метод :doc:`/events/temp_editing/on_cancel_edit`.
