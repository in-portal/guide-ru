OnSetSorting
============
`Data Source`_

.. include:: /includes/not_finished.rst

Устанавливает или изменяет сортировку по колонкам на шаблоне списка записей в административной консоли.

Вызывается из шаблона
---------------------

Вызывается на шаблоне списка записей в административной консоли.

Потенциальное применение
------------------------

Устанавливает сортировку по колонке (так же поддерживается сортировка по двум колонкам) на шаблоне списка
записей в административной консоли.
