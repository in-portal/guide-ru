OnAutoRefreshToggle
===================
`Data Source`_

.. include:: /includes/not_finished.rst

Данное событие используется для установки значения автоперезагрузки списка. Значение автоперезагрузки сохраняется
в базу на долгосрочной основе.

Вызывается из шаблона
---------------------

Данное событие вызывается при установки/снятия галочки "Активный" раздела "Автообновление" в меню "View".

Потенциальное применение
------------------------

Данное событие может быть использовано для обработки "PersistantSession" переменных списка.

Ограничения
-----------

Данное событие будет вызвано, только в том случае, если в конфигурационных переменных установлено значение
``AutoRefreshIntervals``.
