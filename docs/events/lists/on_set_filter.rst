OnSetFilter
===========
`Data Source`_

.. include:: /includes/not_finished.rst

Данное событие вызывается из шаблона на списке записей в административной консоли чтобы установить фильтрацию
по какому-либо полю (или по нескольким полям).

Вызывается из шаблона
---------------------

Вызывается из шаблона списка записей (``grid``) в административной консоли.

Потенциальное применение
------------------------

Используется для установки фильтров на шаблонах списков в административной консоли.
