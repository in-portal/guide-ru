OnMassClone
===========
`Data Source`_

.. include:: /includes/not_finished.rst

Событие вызывается при массовом копировании записей в административной консоли. На самом деле просто подготавливает
список идентификаторов записей, который передаётся в метод ``kTempTablesHandler::CloneItems``, где каждая из записей
копируется по отдельности. Использует вспомогательный метод :doc:`/events/live_editing/store_selected_ids`.

Вызывается из шаблона
---------------------

Вызывается из шаблона в административной консоли после нажатия соответствующей кнопки. Обычно это страница списка записей.

Вызывает события
----------------

- ``kTempTablesHandler::CloneItems``

Потенциальное применение
------------------------

Копирование записей в административной консоли. Например, копирование записей о пользователях.
