OnMassDelete
============
`Data Source`_

.. include:: /includes/not_finished.rst

Событие используется для одновременного удаления двух и более записей в административной консоли.
Использует метод ``kTempTablesHandler:DeleteItems``.

Вызывается из шаблона
---------------------

Вызывается со страницы списка записей в административной консоли после выделения ненужных записей и нажатия
соответствующей кнопки.

Вызывает события
----------------

- ``kTempTablesHandler:DeleteItems``

Потенциальное применение
------------------------

Удаление более чем одной записи из административной консоли.
