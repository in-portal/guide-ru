Стандарты в программировании
****************************
`Data Source`_

Область применения
==================

.. image:: /images/Php_logo.gif
   :width: 120px
   :alt: PHP Logo

В статьях из этой категории описаны стандарты программирования на PHP при использовании Kernel4.
Стандарт разработан на основе
`Zend Framework PHP Coding Standard <http://framework.zend.com/manual/en/coding-standard.html>`__.

Цель
====

Правильные стандарты в программировании важны при разработке любого проекта, а тем более когда над проектом
одновременно работают несколько разработчиков. Наличие стандарта программирования позволяет гарантировать
высокое качество кода, меньшее количество ошибок и лёгкость его поддержки в дальнейшем.
